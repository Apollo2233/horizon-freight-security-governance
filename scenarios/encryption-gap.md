# Encryption Gap – Critical Risk

## Risk Description
Two mission-critical assets were discovered operating without disk encryption enabled.

## Affected Systems
- File Server FS-01
- Warehouse Database Server DB-01

## Security Impact
Unencrypted systems expose sensitive company data to potential theft if the system is compromised or physically accessed.

## Risk Level
Critical

## Recommended Mitigation
- Enable full disk encryption
- Apply encryption policy via endpoint management
- Verify compliance through asset inventory scans

## Security Framework Alignment
NIST 800-53: SC-12 Cryptographic Key Establishment
ISO 27001: A.10 Cryptography
